# pink-room主题介绍
我发现思源的集市上没有粉色系的主题，于是打算自己做一个给自己用。<br>
pink-room的界面颜色以浅粉色、白色以及米色为主，试图营造出清新、温馨的感觉。

主题所需要的**仓耳与墨W02字体**请前往[仓耳字库官网](http://tsanger.cn/product/190)自行下载
